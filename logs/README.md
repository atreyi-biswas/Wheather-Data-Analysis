# Logs (logs)

## Purpose
This folder contains execution logs generated during runtime of the weather pipeline.

## Contents
- weather_fetcher.log → Logs include:
  - API request status
  - Success and failure messages
  - Retry attempts
  - Errors and warnings
  - Execution flow tracking

## Role in Project
This is the monitoring and debugging layer of the project. It helps track system behavior and identify issues during execution.
